Description

This page contains a link to python projects designed for beginners. The purpose of these projects is to allow beginners to learn the fundamentals themselves through practice instead of theory. Each project will focus around a principal feature. It is recommended that you complete these in ascending order, as they are listed in ascending difficulty.

Note:

It is recommended that you do the following before you begin the projects:

  • Read the 'tips' page,

  • Familiarize yourself with the resources listed in the resource page.
