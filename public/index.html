Beginner python projects

This site was made to help beginners learn the Python programming language. Even if you are a complete beginner with no prior coding experience, this should be able to guide through the beginner stages of learning Python. This site will show you how to install the necessary software in order to begin programming, as well as provide projects that are beginner friendly and will help you learn the fundamentals of the language. It is recommended to visit the tips page before you start the projects.

Projects

The main point of this site is to provide you with projects that will help you get learn Python. I've found that the hardest part of learning to code is to know what to code. As you learn more about a coding language, chances are you will think of projects you can do that are possible with that language. The difficulty of these projects will raise as you progress.
