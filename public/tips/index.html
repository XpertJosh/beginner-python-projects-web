Tips

The following is a list of tips that will be useful for beginners:

  • Don't be afraid to search answers on the internet:
    When coding, the majority of your time is often spent thinking of a solution to solve a problem,
    not writing the code to solve the problem. Not even professional programmers remember everything
    they need to know. Don't be afraid to look up how to do something or what something does.
    A popular site for answers is StackOverflow, which can be found on the resources page.

  • Read documentation & reference sites:
    Reference and documentation sites are sites that contain information necessary to code for the
    languages or services they are made for. An example of a reference site is W3Schools, which
    contains lots of information about many languages (including python). Some reference or
    documentation sites also have tutorials, such as W3Schools. Some reference sites can be
    found at the resources page.

  • Try creating your own solutions:
    When you are writing code for a project, try creating your own solution before you look up how to
    accomplish something. This is because you will learn more by coding a solution you created compared
    to coding a solution you found online. This said, don't be afraid to look up a solution, as long as you
    have already tried creating your own solution.

  • Don't get stuck in tutorial hell:
    Tutorial hell is when a beginner programmer only follows tutorials, and as such doesn't learn how to
    code themselves, but rather how to follow a tutorial. While tutorials are useful, they shouldn't be
    overused. For a more detailed explanation of what it is and how to avoid it, click here.
